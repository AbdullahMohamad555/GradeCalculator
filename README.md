# Grade Calculator - Flutter App 🎓

A beautiful and simple grade calculator application built with Flutter, with 100% test coverage.

## ✨ Features

- **Letter Grade Converter**: Convert percentage to letter grade (A, B, C, D, F)
- **Pass/Fail Checker**: Check if a grade is passing (>= 60%)
- **Average Calculator**: Calculate average of multiple grades
- **Beautiful UI**: Modern Material Design 3 interface
- **Error Handling**: User-friendly error messages
- **Full Test Coverage**: Comprehensive unit tests
